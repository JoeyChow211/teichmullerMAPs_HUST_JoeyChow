# teichmullerMAPs_HUST_JoeyChow
**Jacobian Determinant Validation for Lofted Hexahedral Macro-Elements** is a lightweight MATLAB code package for checking the local validity of three-dimensional lofted physical macro meshes.

I have cited the code from the author of the following article, and I would like to express my gratitude here.

[1] P. T. Choi and L. M. Lui, 
     "Fast Disk Conformal Parameterization of Simply-Connected Open Surfaces."
     Journal of Scientific Computing, 65(3), pp. 1065-1090, 2015.
